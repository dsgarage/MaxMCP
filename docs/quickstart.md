# MaxMCP クイックスタート

## 1. インストール

`MaxMCP` フォルダを `~/Documents/Max 9/Packages/` に配置

## 2. セットアップ

```bash
cd ~/Documents/Max\ 9/Packages/MaxMCP
bash setup.sh
```

## 3. Claude Code 設定

setup.sh に表示される JSON を `~/.claude.json` に追加

## 4. 接続

Max 9 → Extras → MaxMCP_Client → connected を確認

## 5. 使い方

Claude Code で話しかけるだけ:
- 「440Hz のサイン波を鳴らして」
- 「減算シンセを作って」
- 「cycle~ って何？」
