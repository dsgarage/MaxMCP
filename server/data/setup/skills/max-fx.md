# /max-fx — エフェクトチェーン構築

ユーザーの説明に基づき、エフェクトチェーンを構築します。

## 手順

1. `max.search.pattern` でエフェクトパターンを検索
2. `max.patcher.info` で現在のパッチ状態を確認
3. 入力段（adc~ または既存オブジェクトから）を設定
4. エフェクトオブジェクトを直列または並列に配置
5. Dry/Wet ミックスとバイパス機能を追加
6. 出力段（dac~ または次段）に接続

## エフェクト構成パターン

| パターン | 構成 | 用途 |
|---------|------|------|
| 直列 | A → B → C | 順番にエフェクト適用 |
| 並列 | A ┬→ B ┬→ Mix | 複数エフェクトを混ぜる |
| センド/リターン | Dry + Send → FX → Return | DAW 的なルーティング |

## 引数: $ARGUMENTS

ユーザーの説明: "$ARGUMENTS"

## 重要ルール
- 各エフェクトに Dry/Wet コントロール（*~ + +~）を付けること
- バイパス用の gate~ または selector~ を入れること
- フィードバックループには oneshot~ または clip~ で発散防止を入れること
- gain~ でレベル管理すること
