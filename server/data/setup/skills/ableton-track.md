# /ableton-track — トラック構成

ユーザーの説明に基づき、Ableton Live のトラック構成を作成します。

## 手順

1. `max.patcher.info` で現在の接続状態を確認
2. ユーザーの説明からトラック構成を設計
3. 各トラックを作成（MIDI / Audio / Return）
4. デバイスを配置（インストゥルメント / エフェクト）
5. ルーティングを設定（入出力 / センド）
6. ミキサー初期値を設定（ボリューム / パン）
7. トラック名・色を設定

## テンプレート種別

| 種別 | トラック構成 | デバイス例 |
|------|------------|----------|
| ドラムキット | MIDI × 1 | Drum Rack |
| ベースライン | MIDI × 1 | Analog / Wavetable |
| シンセパッド | MIDI × 1 | Wavetable / Drift |
| オーディオサンプル | Audio × 1 | Simpler / Sampler |
| ボーカル | Audio × 1 | EQ / Compressor |
| センドリバーブ | Return × 1 | Reverb |
| センドディレイ | Return × 1 | Delay |
| サイドチェイン | Audio × 1 | Compressor（SC入力） |

## 引数: $ARGUMENTS

ユーザーの説明: "$ARGUMENTS"

## ルーティングパターン

| パターン | 説明 |
|---------|------|
| 直列 | トラック → マスター |
| センド/リターン | トラック → Send → Return → マスター |
| グループ | 複数トラック → Group → マスター |
| サイドチェイン | トラックA の信号でトラックB のコンプを駆動 |
| リサンプリング | 出力を別 Audio トラックで録音 |

## 重要ルール
- トラック名は内容がわかる名前を付けること
- トラックの色はセクションごとに統一すること
- センドトラックにはリバーブ・ディレイ等の空間系エフェクトを配置すること
- マスタートラックにはリミッターを入れること
- MIDI トラックにはインストゥルメントを必ず配置すること
- ゲインステージングを意識し、各トラックのレベルを適切に設定すること
