# /ableton-clip — クリップ制作

ユーザーの説明に基づき、MIDI クリップにノートを配置します。

## 手順

1. 現在のトラック・クリップ状態を確認
2. ユーザーの説明からノートパターンを設計
3. MIDI クリップを作成（長さ・ループ設定）
4. ノートを配置（ピッチ・位置・デュレーション・ベロシティ）
5. クオンタイズ設定を適用
6. ループポイントを設定
7. クリップ名を設定

## テンプレート種別

| 種別 | ノート配置 | 典型的な長さ |
|------|----------|------------|
| キックパターン | C1 に 4分音符 | 1-2小節 |
| ハイハット | F#1 / G#1 に 8分 or 16分 | 1小節 |
| スネア | D1 に 2拍目・4拍目 | 1小節 |
| ベースライン | ルート + 5th 中心 | 2-4小節 |
| コード進行 | 3-4音の和音 | 4-8小節 |
| アルペジオ | 和音構成音を順次 | 1-2小節 |
| メロディ | スケール音で構成 | 4-8小節 |

## MIDI ノート配置の基本

| パラメータ | 説明 | 範囲 |
|-----------|------|------|
| pitch | MIDI ノート番号 | 0-127（C3=60） |
| position | 拍内の位置 | 0.0 = 1拍目 |
| duration | ノートの長さ | 拍単位（1.0 = 1拍） |
| velocity | 強さ | 0-127 |

## 引数: $ARGUMENTS

ユーザーの説明: "$ARGUMENTS"

## 重要ルール
- ドラムパターンは General MIDI マッピングを基準にすること（C1=キック, D1=スネア 等）
- ベロシティにバリエーションを付けること（機械的にならないように）
- ループ長はユーザー指定がなければ音楽的に自然な長さにすること
- コード・スケールはユーザー指定のキーに従うこと（指定なしは C マイナー）
- スウィングやヒューマナイズが必要な場合はグルーヴ設定を追加すること
- クリップ名にパターン内容がわかる名前を付けること
