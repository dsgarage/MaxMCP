# /ableton-mixdown — ミックスダウン

ユーザーの説明に基づき、トラックのミックスバランスを調整します。

## 手順

1. 現在のトラック構成とデバイスを確認
2. ユーザーの方針からミックス戦略を設計
3. 各トラックのレベルを設定
4. パンニングを設定
5. EQ で周波数帯域の住み分けを調整
6. コンプレッサーでダイナミクスを制御
7. センドレベル（リバーブ / ディレイ）を設定
8. マスタートラックにリミッター / マキシマイザーを配置
9. 全体のバランスを最終確認

## ミックスの基本方針

| 要素 | パン位置 | レベル目安 | EQ ポイント |
|------|---------|-----------|-----------|
| キック | C（センター） | -6 ~ -3 dB | 60Hz ブースト, 400Hz カット |
| スネア | C | -8 ~ -4 dB | 200Hz カット, 3kHz ブースト |
| ハイハット | L/R 30-50% | -12 ~ -8 dB | 300Hz 以下カット |
| ベース | C | -8 ~ -4 dB | 80Hz ブースト, 250Hz カット |
| シンセ/パッド | L/R 20-60% | -12 ~ -6 dB | 低域カット, 必要帯域ブースト |
| ボーカル | C | -6 ~ -2 dB | 200Hz カット, 3-5kHz ブースト |
| リバーブ Return | - | -18 ~ -10 dB | プリディレイ設定 |

## センド設定ガイド

| エフェクト | 用途 | 典型的なセンド量 |
|-----------|------|----------------|
| リバーブ（短い） | ドラム・パーカッション | -20 ~ -12 dB |
| リバーブ（長い） | パッド・ボーカル | -18 ~ -8 dB |
| ディレイ | リード・ボーカル | -24 ~ -12 dB |
| コーラス / フランジャー | シンセ・ギター | -20 ~ -10 dB |

## マスタリング基礎

| 処理 | デバイス | 設定目安 |
|------|---------|---------|
| EQ | EQ Eight | 微調整のみ（±2dB以内） |
| コンプ | Glue Compressor | Ratio 2:1-4:1, 軽めに |
| リミッター | Limiter | Ceiling -0.3 dB |
| ラウドネス | Utility | 目標 -14 LUFS（ストリーミング） |

## 引数: $ARGUMENTS

ユーザーの説明: "$ARGUMENTS"

## 重要ルール
- キックとベースはセンターに置くこと
- 各トラックの低域を EQ でクリーンアップすること（不要な低域はカット）
- マスタートラックで 0dB を超えないこと（ヘッドルーム確保）
- リミッターの Ceiling は -0.3 dB 以下に設定すること
- ミックスは引き算が基本（ブーストより不要帯域のカットを優先）
- ラウドネスの目標値はプラットフォームに合わせること（Spotify: -14 LUFS 等）
- ソロで聴くのではなく、全体のバランスで判断すること
