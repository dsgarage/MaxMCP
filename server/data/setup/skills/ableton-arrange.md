# /ableton-arrange — アレンジ構成

ユーザーの説明に基づき、楽曲のセクション構成を設計・配置します。

## 手順

1. 現在のセッション状態（トラック・クリップ）を確認
2. ユーザーの説明から楽曲構成を設計
3. シーン（セクション）を作成し名前を付ける
4. 各シーンに対応するクリップを配置
5. セクション間のトランジション（フィル・ブレイク）を追加
6. テンポ・拍子の設定
7. シーンの順序を確認

## テンプレート種別

| ジャンル | 典型的な構成 | BPM |
|---------|------------|-----|
| テクノ | Intro → Build → Drop → Break → Drop → Outro | 125-135 |
| ハウス | Intro → Verse → Chorus → Verse → Chorus → Outro | 120-128 |
| ヒップホップ | Intro → Verse → Hook → Verse → Hook → Outro | 80-100 |
| ポップ | Intro → A → B → サビ → A → B → サビ → 大サビ → Outro | 100-130 |
| アンビエント | Intro → Section A → Transition → Section B → Fade | 60-90 |
| ドラムンベース | Intro → Build → Drop → Break → Drop → Outro | 170-180 |

## セクション設計

| セクション | 役割 | 典型的な長さ |
|-----------|------|------------|
| Intro | 導入・要素を徐々に追加 | 8-16小節 |
| Verse / Aメロ | メインテーマ提示 | 8-16小節 |
| Pre-Chorus / Bメロ | サビへの橋渡し | 4-8小節 |
| Chorus / サビ | 最も盛り上がる部分 | 8-16小節 |
| Build | テンションを上げる | 4-8小節 |
| Drop | エネルギー解放 | 16-32小節 |
| Break | エネルギーを下げる | 8-16小節 |
| Outro | フェードアウト・終結 | 8-16小節 |

## 引数: $ARGUMENTS

ユーザーの説明: "$ARGUMENTS"

## 重要ルール
- 各シーンに分かりやすい名前を付けること（例: "1-Intro", "2-Verse"）
- シーンの色をセクションごとに変えること
- セクション間のトランジションを考慮すること（急な切り替えを避ける）
- ドラムフィルやライザーでセクション変化を演出すること
- BPM はジャンルに合った値をデフォルトにすること
- 全体の長さが音楽的に自然になるようバランスを取ること
