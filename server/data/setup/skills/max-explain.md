# /max-explain — オブジェクト解説

指定されたオブジェクトについて詳しく解説します。

## 手順

1. `max.search.object` で対象オブジェクトの情報を取得
2. オブジェクトの機能・用途・パラメータを日本語で解説
3. インレット/アウトレットの詳細を説明
4. 典型的な使用パターンを示す
5. 関連・代替オブジェクトを紹介
6. よくある間違い・注意点を提示

## 引数: $ARGUMENTS

対象オブジェクト: "$ARGUMENTS"

## 出力フォーマット

以下の構成で解説してください:

### 1. 概要
- 何をするオブジェクトか（1-2文）
- どのドメインか（MSP / Jitter / Max Core / RNBO）

### 2. インレット/アウトレット
- 各インレット: 番号、型、役割、ホット/コールド
- 各アウトレット: 番号、型、出力内容

### 3. 主要な引数・属性
- 作成時引数（例: `cycle~ 440` の 440）
- よく使う属性（@パラメータ）

### 4. 使用例
- 最もシンプルな使い方
- よくある組み合わせパターン

### 5. 関連オブジェクト
- 似た機能のオブジェクト
- よく一緒に使うオブジェクト

### 6. 注意点
- 初学者がハマりやすいポイント
- パフォーマンスに関する注意
